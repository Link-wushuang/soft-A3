# 创新亮点

## 1. 证据化动态学习画像

系统不是一次性问卷，而是从自然语言对话和后续练习行为中持续更新画像。每次更新都保存置信度、证据和变更原因，便于学生和教师理解系统判断依据。

## 2. 可视化多智能体协同

资源生成不是单个黑盒调用，而是展示多个智能体的执行轨迹，包括开始/结束时间、状态、warning 和输出摘要。学生能看到“资源是如何被生成和检查的”。

## 3. 知识库约束与防幻觉流水线

所有资源生成绑定课程知识点、参考来源和常见误区。生成后经过 VerifierAgent 与 ContentGuardAgent 双层检查，再落库展示安全状态。

## 4. 学习闭环而非一次性生成

学生提交练习后，EvaluationAgent 给出评分与错因标签，ReflectionAgent 将错因写回画像，再推动后续路径和资源推荐调整，实现“学习-反馈-补救”的闭环。

## 5. 多模态资源组合

同一知识点可生成讲义、Mermaid 思维导图、分层练习、实操案例、拓展阅读和视频分镜，覆盖阅读型、视觉型和实践型学习偏好。

## 6. 教师可运营视角

教师端提供知识库管理和学情分析，包括薄弱点分布、资源类型分布、正确率趋势，帮助教师从系统数据中发现班级共性问题。
