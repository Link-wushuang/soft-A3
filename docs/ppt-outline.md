# PPT 大纲

## 1. 封面

EduPath Agent：基于大模型的个性化资源生成与学习多智能体系统。

## 2. 赛题理解与痛点

展示传统学习资源固定、学生差异大、反馈滞后的问题。

## 3. 解决方案总览

画像驱动、多智能体协同、资源生成、练习评估和闭环补救。

## 4. 系统架构

Vue 3 前端、FastAPI 后端、MySQL 数据库、LLM Provider、Agent Orchestrator。

## 5. 学习画像

8 维画像字段、证据、置信度、更新日志。

## 6. 多智能体资源生成

展示 14 个 Agent 的职责和 trace 可视化。

## 7. 个性化资源样例

讲义、思维导图、练习、实操案例、拓展阅读、视频分镜。

## 8. 学习效果评估

答题评分、错因标签、画像反思更新和补救推荐。

## 9. 教师端能力

知识库 CRUD、薄弱点分析、资源分布、正确率趋势。

## 10. 安全与防幻觉

知识库约束、VerifierAgent、ContentGuardAgent、安全审计日志。

## 11. 讯飞合规与工程实现

Spark provider、mock 离线演示、MySQL、SSE、测试覆盖。

## 12. 总结与价值

突出创新性、实用性、可扩展性和比赛要求覆盖度。
